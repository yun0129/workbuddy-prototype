- 新增 `.service-target-card` CSS 类 + `selectTarget(el)` JS 函数

- 点击「自定义时间」弹出底部弹层，Step1选日期→自动跳Step2选整点

- 弹层统一：确认弹窗居中+白底阴影，toast居中自动消失

## 技术踩坑记录
- modal-overlay 从 fixed→absolute 后，需显式给吸底弹层加 `align-items:flex-end`
- 吸底弹层需加 `background:#fff` 内联样式，否则背景透明
- 时间弹层：flex-end 定位时 padding-top 在内容区撑开不起作用；最终用 `padding:20px 20px 16px` 解决整体边距，底部确认区 `margin-top:14px` 往上靠
